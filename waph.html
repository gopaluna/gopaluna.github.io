Web Application Programming and Hacking

Front-End Web Development Lab

Instructor: Dr. Phu Phung

Overview of this course

In this course we are learning the basics of web application development along with the vulnerabilities a web page can contain. Learning these concepts will help us develop applications that are hard to exploit by the hackers.

Projects and Hackathons

In this course, the professor has designed several projects and hackathons that will help us gain hands on experience with the technologies and the concepts we are learning in the class.

The projects include

  • Developing a personal home page
  • A simple Blog System
  • And another team project of our choice

The hackathons include

  • XSS Attacks and Defenses
  • SQL Injection Attacks
  • Session Hi-Jack Attacks
  • Cross Site Request Forgery Attacks
